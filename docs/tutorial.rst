Tutorials
==============================

To create and perform your own experiment you should prepare the data, following the :ref:`data_proc_tutorial`.

After the data is prepared, create and customize the configuration file to your needs, following the
:ref:`config_tutorial`.

For the local experiment refer to :ref:`local_comm_tutorial`.

And to launch distributed multinode or multiprocess experiment go to :ref:`distr_comm_tutorial`.
